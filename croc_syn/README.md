# CROC SYN
These folders can be used for dataset generation in CROC_syn style. Each subfolder contains commented code and/or an additional readme file. 

```
1. prompt_gen contains scripts to generate taxonomy based contrastive prompts
2. img_gen uses these outputs to generate respective images
(2.5 metrics/apply_metrics.sh on the top level can be used to get metric scores for these outputs)
3. eval_and_plot evaluates metric scores generated for crocsyn
```

Due to its size, please contact us to get access to the full CROC_syn dataset.
